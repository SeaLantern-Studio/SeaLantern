# Security Policy | 安全策略

## Supported Versions | 支持的版本

- [x] Nightly Build: Very Fast
- [x] Release (Latest): Fast

## Reporting a Vulnerability | 报告漏洞

### Level | 等级

You can select an level what you want report.

你可以选择一个等级作为你想报告漏洞的等级

> `Super` mean we need the fastest speed to fix this vulnerability, like it was found who use the vulnerability to inject somebody's server.
>
> 该等级意味着已经存在在野利用的安全危机.

- [ ] Super

> `High` mean we need fastly to fix this vulerability, like key leak, some high performance questions.
>
> 该等级意味着一些很可能被利用的高危漏洞,但没有在野利用或尚未确认,需要紧急修复.

- [ ] High

- [ ] Medium

- [ ] Low
