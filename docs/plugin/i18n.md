## sl.i18n

本文档说明插件运行时暴露的 [`sl.i18n`](../../backend/tauri-host/src/plugins/runtime/i18n/mod.rs) Lua 接口，用于查询当前语言、读取翻译、监听语言切换，以及注册插件自己的国际化资源。

## APIs

- `sl.i18n.getLocale()`: 获取当前应用语言代码，例如 `zh-CN`、`en-US`
- `sl.i18n.t(key, options?)`: 按 key 获取翻译文本；当提供 `options` 时会进行 `{name}` 风格变量替换
- `sl.i18n.hasTranslation(key, locale?)`: 判断指定 key 是否存在翻译；不传 `locale` 时使用当前语言，带 fallback 行为
- `sl.i18n.tOrDefault(key, defaultValue, options?)`: 若 key 存在翻译则返回翻译结果，否则返回默认值
- `sl.i18n.getAllTranslations()`: 获取当前语言下可读到的全部翻译项，包含宿主与插件翻译的合并结果
- `sl.i18n.getTranslations(locale)`: 获取指定语言下可读到的全部翻译项
- `sl.i18n.getAvailableLocales()`: 获取当前可用语言列表
- `sl.i18n.tp(pluginId, key)`: 读取指定插件命名空间下的翻译，相当于查询 `plugins.{pluginId}.{key}`
- `sl.i18n.onLocaleChange(callback)`: 注册语言切换监听器，返回回调 id，用于后续取消监听
- `sl.i18n.offLocaleChange(callbackId)`: 取消指定 id 的语言切换监听器，成功返回 `true`
- `sl.i18n.registerLocale(locale, displayName)`: 注册插件提供的新语言及其显示名称
- `sl.i18n.addTranslations(locale, entries)`: 为插件写入指定语言的翻译项；写入时会自动添加插件命名空间前缀
- `sl.i18n.removeTranslations()`: 移除当前插件此前注册的全部翻译项

## 安全限制

当前写接口已带有基础安全校验，主要包括：

| 限制项            | 规则                                       | 对应实现                                                                                      |
| ----------------- | ------------------------------------------ | --------------------------------------------------------------------------------------------- |
| locale 格式       | 仅允许如 `en`、`en-US`、`zh-CN`、`sr-Latn` | [`validate_locale_input()`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:88)             |
| locale 长度       | 1 到 32 个字符                             | [`MAX_LOCALE_LEN`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:9)                       |
| display name 长度 | 1 到 64 个字符，且不能包含控制字符         | [`validate_display_name()`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:105)            |
| 翻译 key 字符集   | 仅允许字母、数字、`.`、`-`、`_`、`:`       | [`validate_translation_key_input()`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:123)   |
| 翻译 key 长度     | 1 到 128 个字符                            | [`MAX_TRANSLATION_KEY_LEN`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:11)             |
| 翻译 value 长度   | 最多 4000 个字符，且不能包含空字符         | [`validate_translation_value()`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:140)       |
| 单次写入条数      | 最多 500 条                                | [`MAX_TRANSLATION_ENTRIES_PER_CALL`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:13)    |
| 单插件总翻译条数  | 最多 5000 条                               | [`enforce_plugin_translation_quota()`](../../backend/tauri-host/src/plugins/runtime/i18n/write.rs:157) |
| 插件命名空间隔离  | 自动前缀化为 `plugins.{plugin_id}.`        | [`plugin_i18n_namespace()`](../../backend/tauri-host/src/plugins/runtime/i18n/common.rs:67)            |

## 备注

- [`sl.i18n.getAllTranslations()`](../../backend/tauri-host/src/plugins/runtime/i18n/mod.rs:76) 返回的是当前语言下可访问到的翻译集合。
- [`sl.i18n.getTranslations(locale)`](../../backend/tauri-host/src/plugins/runtime/i18n/mod.rs:88) 可用于调试、导出或按语言预览翻译数据。
- 插件监听器在最后一个回调移除时会自动释放底层 token，插件卸载时也会自动清理相关 i18n 资源。
